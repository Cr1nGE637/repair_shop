# Ремонтная мастерская

Система управления ремонтной мастерской бытовой техники.

## Описание

Система предназначена для управления ремонтом бытовой техники физических лиц. Мастерская выполняет следующие функции:

- Прием техники на ремонт с фиксацией описания проблемы со слов клиента
- Управление складом компонентов и материалов
- Закупка компонентов при их отсутствии на складе
- Оценка стоимости ремонта с указанием всех видов работ и использованных материалов
- Формирование актов выполненных работ при выдаче техники
- Хранение истории всех выполненных ремонтов и отказов

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Выполните миграции:
```bash
python manage.py migrate
```

3. Создайте суперпользователя:
```bash
python manage.py createsuperuser
```

4. Запустите сервер разработки:
```bash
python manage.py runserver
```

5. Откройте браузер и перейдите по адресу:
- Главная страница: http://127.0.0.1:8000/
- Админ-панель: http://127.0.0.1:8000/admin/

## Структура проекта

### Модели

- **Client** - Клиенты (физические лица)
- **Device** - Бытовая техника
- **Component** - Компоненты и материалы на складе
- **WorkType** - Виды выполняемых работ
- **Repair** - Ремонты/заказы
- **RepairWork** - Выполненные работы в ремонте
- **RepairComponent** - Использованные компоненты в ремонте
- **RepairAct** - Акты выполненных работ

### Основные функции

1. **Управление клиентами и техникой**
   - Регистрация новых клиентов
   - Добавление техники клиента
   - Просмотр истории ремонтов по клиенту

2. **Управление складом**
   - Учет компонентов и материалов
   - Отслеживание остатков
   - Уведомления о низких остатках

3. **Управление ремонтами**
   - Прием техники на ремонт
   - Отслеживание статусов ремонта
   - Добавление выполненных работ
   - Учет использованных компонентов
   - Автоматический расчет стоимости

4. **Акты выполненных работ**
   - Автоматическое формирование актов
   - Печать актов
   - Полная смета работ и материалов

## Использование

### Рабочий процесс

1. **Прием техники:**
   - Создайте или выберите клиента
   - Добавьте технику клиента
   - Создайте новый ремонт с описанием проблемы

2. **Выполнение ремонта:**
   - Измените статус на "В работе"
   - Добавьте выполненные работы
   - Добавьте использованные компоненты (со склада или закупленные)
   - Система автоматически рассчитает стоимость

3. **Выдача техники:**
   - Измените статус на "Готов к выдаче" или "Выдан"
   - Создайте акт выполненных работ
   - Распечатайте акт для клиента

### Статусы ремонта

- **Принят** - Техника принята на ремонт
- **В работе** - Ремонт выполняется
- **Ожидание запчастей** - Ожидание поступления компонентов
- **Готов к выдаче** - Ремонт завершен, готов к выдаче
- **Выдан** - Техника выдана клиенту
- **Отменен** - Ремонт отменен
- **Невозможно отремонтировать** - Ремонт невозможен

## Админ-панель

Админ-панель Django предоставляет полный доступ ко всем моделям:
- Управление клиентами, техникой, компонентами
- Создание и редактирование ремонтов
- Добавление работ и компонентов к ремонтам
- Формирование актов выполненных работ
